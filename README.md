# Packet Sniffer (Python + Scapy)

## 🚀 Features
- Real-time packet capture
- TCP/UDP parsing
- HTTP traffic filtering
- Lightweight & extensible

## 🛠️ Tech Stack
- Python
- Scapy

## ▶️ Usage
Run as root (required for packet capture):

sudo python sniffer/main.py

## ⚠️ Disclaimer
This tool is for educational purposes only. Use only on networks you own or have permission to monitor.
